Connected via the Satori protocol, you need to select ``Satori`` as the adapter.

# LuckyLillia

- [LuckyLillia](https://github.com/LLOneBot/LuckyLilliaBot) Install the client to use the Satori protocol

LuckyLillia installation tutorial: https://luckylillia.com/guide/choice_install

![Image 1](/images/zh/deploy/bots/satori/llbot_local_setting.png)

Make sure the IP address in the address field is the address of the host running LangBot (use ```127.0.0.1``` if running on the same network). The default port ``5600`` can be changed as needed. Setting a token is recommended.
>
> ![Image 2](/images/zh/deploy/bots/satori/llbot_open_setting.png)
>
> This image shows the public network configuration. Make sure the IP address is the address of the host running LangBot. In this case, you need to expose the corresponding LangBot port.

## LangBot Integration

Next, open the LangBot configuration page.

Click on Bots, then click Add.

Select `Satori` for `Platform/Adapter`.

![Image 3](/images/zh/deploy/bots/satori/llone_langbot_build.png)
> If you use the default configuration (local network), you don't even need to modify any parameters.

After creation, you can still adjust the parameters.

![Image 4](/images/zh/deploy/bots/satori/llone_langbot_setting.png)

Make sure you have enabled this adapter.

![Image 5](/images/zh/deploy/bots/satori/satori_platform_open.png)
