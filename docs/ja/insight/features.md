# 機能

LangBotは、オープンソースのLLMネイティブインスタントメッセージングロボット開発プラットフォームであり、すぐに使えるIMロボット開発体験を提供することを目的としています。Agent、RAG、MCPなどのLLMアプリケーション機能を備え、グローバルなインスタントメッセージングプラットフォームをサポートし、豊富なAPIインターフェースを提供し、カスタム開発をサポートします。

## システムの拡張性

- HTTP Service API: APIキー認証を使用して、HTTP APIインターフェースを通じてLangBotのさまざまなエンティティにアクセスすることをサポートします。
- Outgoing Webhook: Webhookを通じてLangBotの内部イベントを外部システムに送信することをサポートします。
- プロダクショングレードのプラグインシステム: クロスプロセス通信技術と非同期技術に基づいたプロダクショングレードのプラグインシステムで、さまざまなコンポーネント拡張をサポートします。
