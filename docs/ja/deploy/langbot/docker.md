# Dockerデプロイ

:::warning
まず以下について学習することをお勧めします:

 - `Docker`と`Docker Compose`の使用方法
 - Dockerコンテナネットワーク通信の設定方法
:::

:::info
- Git、Docker、Docker Composeがインストールされていることを確認してください
:::

プロジェクトをクローンします:

```bash
git clone https://github.com/langbot-app/LangBot
cd LangBot/docker
```

コンテナを起動します:

```bash
docker compose up
```

コンテナはWebUIアクセス用にポート`5300`をマッピングします。`http://127.0.0.1:5300`にアクセスしてWebUIを表示できます。
また、メッセージプラットフォームアダプター用に予約されたポート`2280-2290`もマッピングします。

LangBotのデプロイ完了後、引き続き以下をお読みください:

- [ボット設定ドキュメント](/en/deploy/platforms/readme)
- [モデル設定ドキュメント](/en/deploy/models/readme)
