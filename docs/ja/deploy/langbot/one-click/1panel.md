# 1Panelデプロイ

[1Panel](https://www.bt.cn/new/download.html?r=dk_LangBot)を使用してLangBotをワンクリックでデプロイします。

:::warning

追加のコンテナを必要としないメッセージプラットフォーム(DiscordやLarkなど)を使用する予定がない場合、または1Panel上のDockerコンテナ間のネットワーク接続に精通していない場合は、1Panelのワンクリックデプロイは推奨されません。代わりに[Dockerデプロイ](/en/deploy/langbot/docker)の使用を検討してください。

:::

:::info

[Alibaba Cloud](https://www.aliyun.com/minisite/goods?userCode=ys4ad8gs)なら、年間38元という低価格でサーバーを入手でき、さらに20%の割引も受けられます。1Panelをワンクリックでインストールするのがより便利です。

:::

## 1Panelのインストール

> 推奨サーバーシステム(1Panelより): Debian 12、Ubuntu 22など

この記事では、サーバーにデフォルトでパブリックIPv4アドレスがあることを前提としています。パブリックIPのないホームマシンを使用している場合は、以下で説明する内部ネットワーク経由で関連ページにアクセスできます。

[1Panel](https://www.bt.cn/new/download.html?r=dk_LangBot)にアクセスし、1Panelが公式に提供するLinuxパネルインストールスクリプトを使用します。お使いのオペレーティングシステムに応じて対応するスクリプトを選択してください。

![1Panelのインストール](/assets/image/zh/deploy/langbot/one-click/1p_install_01.png)

対応するスクリプトをコピーし、サーバーに貼り付けて実行します。

しばらく待つと、1Panelのインストールが完了します。

次に`外部アドレス`をコピーしてブラウザで開きます。

## LangBotのインストール

`アプリストア`を開き、`AI`カテゴリを選択し、LangBotをクリックしてインストールします。

![LangBotのインストール](/assets/image/zh/deploy/langbot/one-click/1p_langbot_01.png)

`外部ポートアクセス`のチェックボックスをオンにして、確認をクリックします。

![LangBotのインストール](/assets/image/zh/deploy/langbot/one-click/1p_langbot_02.png)

しばらく待つと、インストールが完了します。

左側の`コンテナ`をクリックし、`コンテナ`カテゴリを選択すると、LangBotコンテナバーが表示されます。

![LangBotコンテナ情報の表示](/assets/image/zh/deploy/langbot/one-click/1p_langbot_03.png)

サーバーのパブリックIPアドレスをコピーします。例: `http://xxx.xxx.xxx.xxx:5300`(xxx.xxx.xxx.xxxは1Panelが配置されているサーバーのIPアドレス、5300はポート番号)。

アドレスを開き、管理者のメールアドレスとパスワードを入力して、初期化をクリックします。

その後、ビジュアルインターフェースでLangBotを設定できます。

次に、[メッセージプラットフォームのデプロイ](/en/deploy/platforms/readme)ページを引き続きお読みください。
