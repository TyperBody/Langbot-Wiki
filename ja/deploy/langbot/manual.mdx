## メインプログラムのインストール

1. [リリース](https://github.com/langbot-app/LangBot/releases)ページにアクセスし、最新バージョンのzipパッケージ(推奨)`langbot-xxx-all.zip`をダウンロードします(Source Codeはダウンロードしないでください。WebUIが含まれていません)。展開して、展開されたディレクトリでコマンドライン(ターミナル)を開きます。

![リリースのダウンロード](/images/zh/deploy/langbot/manual/dl_release.png)

2. 依存関係のインストール

依存関係の管理には[uv](https://docs.astral.sh/uv/)を使用します。

```bash
pip install uv
uv sync
```

3. メインプログラムを一度実行して設定ファイルを生成します

```bash
uv run main.py
```

以下のメッセージが表示されます:

```
 _                   ___      _
| |   __ _ _ _  __ _| _ ) ___| |_
| |__/ _` | ' \/ _` | _ \/ _ \  _|
|____\__,_|_||_\__, |___/\___/\__|
               |___/

⭐️オープンソース: https://github.com/langbot-app/LangBot
📖ドキュメント: https://docs.langbot.app

以下のファイルが存在しないため、自動的に生成されました。必要に応じて設定ファイルを変更して再起動してください:
- data/config.yaml
...
```

Ctrl+Cでプログラムを終了し、[ボットのデプロイ](/en/deploy/platforms/readme)ページに進んでください。
