# LuckyLillia

- [LuckyLillia](https://github.com/LLOneBot/LuckyLilliaBot) 安装客户端以使用Satori协议

安装LuckyLillia的教程：https://luckylillia.com/guide/choice_install

![图片1](/images/zh/deploy/bots/satori/llbot_local_setting.png)

确保地址中 IP地址 是 LangBot 运行的主机的地址（若在同一网络内运行则为 ```127.0.0.1```） 默认端口 ``5600`` 可以根据需要更改 推荐设置令牌
>
> ![图片2](/images/zh/deploy/bots/satori/llbot_open_setting.png)
>
> 该图为公网配置确保地址中 IP地址 是 LangBot 运行的主机的地址 此时配置需要暴露langbot的该端口

## LangBot 对接

接下来打开 LangBot 配置页面

点击机器人，点击添加

`平台/适配器选择`为`Satori`

![图片3](/images/zh/deploy/bots/satori/llone_langbot_build.png)
> 如果你使用默认配置(内网)你甚至不用修改任何参数

创建好后你仍然可以调整参数

![图片4](/images/zh/deploy/bots/satori/llone_langbot_setting.png)

确保你启用了本适配器

![图片5](/images/zh/deploy/bots/satori/satori_platform_open.png)

<Info>
注意，如果涉及多个 docker 容器的网络连接问题，请参考 [网络配置详解](/zh/workshop/network-details)
